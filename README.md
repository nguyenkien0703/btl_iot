## Setup steps for the project

```
npm i -g yarn
```
```
$ yarn install

```
docker compose up -d
```
```
yarn run start:dev
```



## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
